## About

Beginner-friendly recycling side hustle with foreman NPC, sorting, and cash payouts.

Clock in at the yard, grab boxes, sort recyclable material, and sell for cash. Supports ESX, QBCore, and Qbox with ox_lib menus, carry animations, and configurable locations and payouts.

## Features

- Foreman NPC clock-in system
- Box sorting gameplay
- Cash payouts
- Configurable yard locations
- Carry animations

## Requirements

- ox_lib
- ESX Legacy / QBCore / Qbox
- ox_inventory recommended
